5. What are different types of lists in HTML ? With Example

Ans : there are three types of list in HTML.

-ordered list

An ordered list is represented by (ol) tag is used to create a list by numberical or alphabetical indicator.the (ol) tag is paired with (li) list item tag to define each individual item.

Example :

  1. First item
  2. Second item
  3. Third item

-unordered list

An unordered list is represented by (ul) tag is used to create a list of item by a bullet point and other marker.the (ul) tag is paired with (li) list item tag to define each individual item.

Example :

  • First item
  • Second item
  • Third item

-description list

A description list is a way to display a list of terms with their corresponding descriptions. It is represented by the (dl) tag (description list), which contains one or more pairs of (dt) (description term) and (dd) (description details) elements.
