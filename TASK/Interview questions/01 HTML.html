INTERVIEW QUESTIONS

01. What does HTML stands for ?

Ans : Hypertext Markup language or html is a markup language that is used to create website templates or webpages to present the content on the world wide web.

02. What are HTML attributes ?

Ans : Attributes are the properties that can be added to an html tag.these attributes change the way the tag behaves or its displayed.

Example : img src="" alt=""

03. Name some new features which were not present in html but are added to html5 ?

Ans : Some new features in HTML5 include:

-DOCTYPE declaration : !DOCTYPE html

-section

-header

-article

-aside

-Nav

04. Define semantic elements in html.

Ans : semantic elements are the HTML element that represent its meaning to the browser and developer about its contents.

05. meta tags are used in html ?

Ans : meta tags in html are used by the developer to tell the browser about page descriptions author of the templates character set keywords and many more.
