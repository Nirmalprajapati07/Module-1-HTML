02 Tags and Element

- TAGS

tags are words or letters surrounded by angle brackets they are used to create html elements such as paragraph,links etc.tags are simple instructions that tell a web browser how to format text. tags are two types.

1. opening tag and closing tag eg html,head,body,h1,p etc

2. self closing tag or empty tag or void tag eg meta,img,hr etc

- Elements

elements are those for creating web pages and showing the content on them. An HTML element is made up of a start tag, attributes, end tag, and content between them.

03 headingpara
