1. website
website is a collection of web resources like web pages, multimedia content ( audio,video,animation) which is identified by domain name and located at web server like wwww.google.com
website may be static or dynamic.
static website 
collection of web pages with fixed contents.
dynamic websites 
collection of dynamic pages that changes according to user input,date,time,location etc.
2. www (world wide web)
It is a system of interconnected public webpages accessible through the internet.
